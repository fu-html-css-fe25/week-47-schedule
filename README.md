###### HTML & CSS, vecka 6 av 8

## Introduktion

En bra webbplats funkar för alla – oavsett förmåga, teknik eller förutsättningar. Vi går igenom hur man skriver tillgänglig HTML och CSS, och lär oss tänka inkluderande redan från start. 
Dessutom introduceras BEM, en tydlig namngivningsmetod som hjälper oss att skriva renare och mer organiserad CSS – något du kommer tacka dig själv (och ditt team) för längre fram.

## Mål för veckan:
1. Förstå vad tillgänglighet innebär, samt hur man kontrollerar tillgänglighet
2. Förstå användningsområden för aria-attribut
3. Förstå varför vi behöver namngivningskonventioner när vi utvecklar
4. Förstå hur man applicerar BEM i sin kod
